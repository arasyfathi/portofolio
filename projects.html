Smart Dustbin IoT & Blynk Dashboard

Engineered a smart automated dustbin with real-time fill-level monitoring via Blynk.Console and Blynk.cloud. An ESP32 reads an ultrasonic sensor and drives a servo motor actuator, with hardware status streamed to the Blynk dashboard for remote monitoring.
